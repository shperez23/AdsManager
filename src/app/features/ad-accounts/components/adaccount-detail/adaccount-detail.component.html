Acciones soportadas

  • GET /api/v1/adaccounts

    Consulta el listado usando únicamente los parámetros técnicos Status, Page, PageSize, Search, SortBy y SortDirection.

  • POST /api/v1/adaccounts/import-from-meta

    Importa cuentas desde Meta y refresca el listado del módulo.

  • POST /api/v1/adaccounts/{{ '{' }}id{{ '}' }}/sync

    Sincroniza una cuenta puntual desde el listado principal.
